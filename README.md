# jeckin
HTTP Injector

## Requirement
1. python > 3. < 4
2. on your OS you need to install ssh, sshpass and corkscrew, for Windows user you can install via cygwin

## Installation
```
$ pip install jeckin
```

## Usage
Generate template config file
```
jeckin myconfig.hj
```

Run service
```
jeckin myconfig.hj
```
